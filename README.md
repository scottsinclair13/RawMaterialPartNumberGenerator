# Raw Material Part Number Generator

A simple, user-friendly **Streamlit web application** that generates standardized part numbers for raw materials based on selected categories, material properties, grades, specifications, and other parameters.

This tool ensures consistency in part numbering across engineering, procurement, and manufacturing teams by enforcing predefined rules and options.

## Features

- Interactive dropdowns and selectors for material categories, forms, alloys, tempers, AMS/phenolic specs, cable/plastic grades, and more
- Real-time part number generation using configurable logic
- Clean, centered Streamlit interface with custom page icon and title
- Extensible – easy to add new categories, options, or rules in `data.py`
